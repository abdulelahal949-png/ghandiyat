# مقرأة غِنديات لتعليم القراءات

## نظرة عامة
موقع تعليمي قرآني فاخر لمقرأة غِنديات، تابعة لجمعية طابة بالمدينة المنورة. يُقدّم تعليم القراءات العشر المتواترة للنساء عن بُعد بأسانيد متصلة، بإشراف معلمات مجازات.

- حلقات تثبيت ومراجعة الحفظ
- برامج تأصيل في علم القراءات والرسم والضبط
